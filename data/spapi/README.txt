A simple and hardcoded API used to expose the surface computation of SimaPro
Run on Windows with: `uvicorn.exe --workers 1 --host 0.0.0.0 simapro:api`
You should configure the Windows firewall to open port 8000
Here is the doc to the COM interface of SimaPro: https://pre-sustainability.com/legacy/download/Programming-The-SimaPro-COM-interface-oct2013.pdf
